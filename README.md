# ML-phishing-detection

Este projeto utiliza um conjunto de dados público voltado à detecção de ataques de phishing, um tipo de ameaça em que invasores se passam por entidades confiáveis para induzir usuários a revelar informações sensíveis. O dataset empregado neste estudo está disponível no Kaggle: [Phishing Dataset for Machine Learning](https://www.kaggle.com/datasets/shashwatwork/phishing-dataset-for-machine-learning).

A base reúne 48 atributos extraídos de 10.000 páginas da web, sendo metade delas legítimas e metade classificadas como phishing. Esses dados foram coletados entre 2015 e 2017 utilizando uma técnica aprimorada de extração baseada em automação de navegadores (Selenium WebDriver), o que garante maior precisão e robustez quando comparado a métodos tradicionais baseados apenas em expressões regulares.

O conjunto é especialmente útil para pesquisadores e profissionais de segurança cibernética, permitindo análises de características de páginas maliciosas, experimentação rápida de técnicas de machine learning e comparação entre diferentes modelos de detecção. Um material essencial para quem navega nas fronteiras entre segurança digital e inteligência computacional. 

Um outro conjunto que pode ser explorado é: [PhiUSIIL Phishing URL Dataset ](https://archive.ics.uci.edu/dataset/967/phiusiil+phishing+url+dataset). Ele oferece atributos complementares para enriquecer a análise e ampliar a robustez dos experimentos conduzidos no projeto.
